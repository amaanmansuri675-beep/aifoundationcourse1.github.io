DAY-3

Building & Deploying a Website Using ChatGPT and GitHub Pages
  • 1. Generate website code with ChatGPT
    Prompt: "Create a simple HTML page with title, heading, and paragraph."
  • 2. Save your code as index.html
    Ensure the file name is exactly "index.html".
  • 3. Create a GitHub account
    Visit github.com, sign up, and confirm your email.
  • 4. Create a new repository
    Click "New Repository" → Name it "yourusername.github.io" → Keep public.
  • 5. Upload index.html
    Inside repository → Add file → Upload files → Select index.html.
  • 6. Commit your changes
    Scroll down and click "Commit changes".
  • 7. Check your live website
    Visit "yourusername.github.io" in your browser.
Common Mistakes to Avoid
  • File name must be index.html.
  • Repository name must be lowercase.
  • Always commit changes after upload.
Deploy Updates

To update your site, edit index.html and commit changes again.

Final Project

Every student must create a personal webpage (e.g., an intro page about hobbies, skills, or profile).
